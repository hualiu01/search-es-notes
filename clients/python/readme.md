# Get Started
__prerequisite__ 
1. have a ES cluster running
2. config a `.env` with the following
```
ES_ENDPOINT =
ES_USERNAME =
ES_PASSWORD = 
```
__venv__
create .venv and install requirements 
```
make
```
To run all the tests:
```
make test
```

__For Developers__
To reformat
```
make format
```
